### 동적 메모리 할당 및 반환

new 연산자와 delete 연산자

c++의 동적 메모리 할당과 반환은 new와 delete 연산자로 가능하다.

포인터 변수로 할당 해야 하고 반환을 해주지 않으면 메모리 누수가 발생한다.

동적 메모리를 배열로 할당 받을 수 있고 이 때 반환은 delete []을 사용해야 한다.

동적 메모리로 객체를 할당 받고 반환할 수 있다. 객체를 할당 받을 때는 기본 생성자 함수가 실행된다.

정적 메모리 할당 -> 컴파일할때

동적 메모리 할당 -> 런타임할때

힙은 프로세스가 실행을 시작할 때 운영체제가 할당 해 준다.

### this 포인터 변수

this는 자기 자신의 객체의 위치를 나타내는 매개변수이다.

컴파일러가 멤버 함수를 컴파일 할 때 만약 ob.setA(5);라는 함수 호출이 있다면

이렇게 해석한다. setA(&ob,5); 이 때 ob가 참조로 this라는 포인터 변수로 호출 된다.

범위 규칙 같은 이름의 변수는 가장 가까운 위치의 이름으로 판정한다. 따라서 

매개 변수의 이름과 멤버 변수의 이름이 같다면 this를 통해 멤버 변수를 가리킬 수 있게 된다.

또한 객체 자기 자신을 리턴해야 할 때 this를 사용해 객체 자기 자신을 반환 할 수 있다.

return *this (this는 포인터 변수기 때문에 *)
